Description

pcapstreamer is a network packet dumper and parser which uses libpcap to capture packets.

Options

-l|--listinterfaces

list all interfaces available for packet capture. If you do not get any interface, probably you do not have permission to capture. You need to be root to capture packets.

-i|--interface

name must be an interface name resulted from --listinterfaces. This will set the interface to capture. Packets will be retrived going through this interface.We can use default to capture packets from default interface determined by libpcap or we can use null to capture from linux's pseudo interface. We must provide an interface to pcapstreamer to capture packets. Press ctrl-c to stop capturing packets.

-p|--parse

This option will try to parse the packet captured by libpcap. Header fields will be seperated by comma. By default pcapstreamer will dump packets in binary strings.

-h|--help

This option will display short usage.

pcap_expression

libpcap expression to filter packets during capture time. See pcap-filter(7) for more details.

Examples

Following command will list all interfaces available for capturing. 

pcapstreamer -l

Following command will capture packets going through lo interace.

pcapstreamer -i lo

Following command will capture packets going through Linux's pseudo interface with parsing enabled.

pcapstreamer -i null -p
